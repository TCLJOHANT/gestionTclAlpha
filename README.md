## Gestion Tcl Alpha

### Requisitos Instalación

- Docker

### Instalación

1. Clonar el repositorio
2. Ejecutar los siguientes comandos en la terminal

```bash
    cd gestionTclAlpha
    docker-compose up -d
```

3. Abrir la aplicación en el navegador

```bash
    http://localhost:80/admin
```
